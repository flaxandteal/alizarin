## Installing Alizarin

```sh
npm install alizarin
```

At present, `alizarin` has not reached a stable distribution, so you
will likely prefer to check out our [Github repo](https://github.com/flaxandteal/alizarin).

## Running Tests

```bash
npm test
```

## Linting

```bash
npm run lint
```
